安装xlrd

1. 从https://pypi.python.org/pypi/xlrd 下载合适版本

下载安装包点击该处：[xlrd-0.9.4.tar.gz]

2.解压下载的包到指定目录

3.在cmd中cd到解压目录下，执行python setup.py install


4.验证

未安装前import模块会报错，安装后倒入模块不报错，表示安装成功！
