批量解压

发表于 2019-05-28 | 更新于 2019-05-31 | 分类于 系统 ， Linux ， 命令
本文字数： 890 | 阅读时长 ≈ 1 分钟

昨天有个需求：将一个目录下的所有tar.gz文件解压。

方法一：

for tar in *.tar.gz;do tar xzvf $tar;done

方法二：

ls *.tar.gz | xargs -n1 tar xzvf

解压到指定目录

ls *.tar.gz | xargs -n1 tar -C tmp -xzvf
