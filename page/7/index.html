PMP:绩效测量基准

发表于 2019-07-31 | 分类于 PMP ， 知识点
本文字数： 632 | 阅读时长 ≈ 1 分钟

绩效测量基准：把范围、进度、成本三位一体的综合基准。用来和项目的实际情况进行比较。

范围也由三个内容构成：范围说明书、wbs和wbs词典。

进度基准是指一段时间包含了应急储备不包括管理储备。

成本基准汇总所有活动的成本估算，加上应急储备。成本基准也是包含了应急储备不包含管理储备，如果需要动用管理储备需要走完整的变更管理流程。把动用的那部分管理储备添加到成本基准当中才可以使用。

2019.7.30每日一题：

A project manager finalizes a project scope statement and realizes it is similar to one completed last year. Based on last year’s project, what should the project manager reuse?
A. Project charter
B. Risk management plan
C. Project budget
D. Work breakdown structure (WBS)

项目经理最终确定项目范围说明书，并意识到该项目与去年完成的一个项目类似。根据去年的项目，项目经理应该重复利用下列哪一项？
A.项目章程
B.项目管理计划
C.项目预算
D.工作分解结构（WBS）

答案：D
解析：本题属于组织过程资产的应用，在确定范围说明书如果可以确定与过去项目类似，接下来的WBS则可以参照和利用过往的模板。虽然A、B参考答案中，项目章程和项目管理计划也属于组织过程资产，但是题意说明当前结点是在确定范围说明书后，即章程也已经完成了；而项目管理计划则范围太大，最切合题意的是参考答案D。

PMP:焦点小组和名义小组

发表于 2019-07-30 | 分类于 PMP ， 知识点
本文字数： 832 | 阅读时长 ≈ 1 分钟

名义小组技术一般用在头脑风暴之后。先使用头脑风暴收集到各种创意，但是头脑风暴不能够做最终的决策，那么名义小组技术就可以帮助做最终的决策。
步骤：先使用头脑风暴收集各种创意，然后展开各种创意的优先级排序。选出一些对问题有研究有经验的人作为小组成员，给他们提供决策问题的相关信息，这些小组成员事先不通气，请他们独立思考，然后要求他们把自己的决策方案和意见写下来。再请他们一个接一个地陈述自己的意见和看法，让每个人对自己的方案进行投票，投票数最多的就成为最终要决策的方案。
头脑风暴无法做出最终的决策，而名义小组技术通过投票得到最优先级的创意。

焦点小组：由一位受过训练的主持人，引导大家进行互动式的讨论，往往是8-12个人组成的小组，常常有主题专家的参与。

PMP:进度压缩

发表于 2019-07-30 | 分类于 PMP ， 知识点
本文字数： 879 | 阅读时长 ≈ 1 分钟

进度压缩两种方法：赶工和快速跟进

赶工：增加资源，以最小的成本增加来压缩项目的进度工期。（比如说：加班、加人、加钱）
针对的是关键路径，而且是关键路径上最便宜的活动，一定会引起成本的增加。
用钱换时间。

快速跟进：原先按顺序执行的活动现改为并行执行。
多个活动同时做势必会引发风险。
用风险换时间。

PMP:项目进度计划的三种表现形式

发表于 2019-07-30 | 分类于 PMP ， 知识点
本文字数： 1.3k | 阅读时长 ≈ 1 分钟

项目进度计划的三种表现形式：里程碑图、横道图、逻辑横道图。

里程碑图，显示出了主要可交付成果所有的里程碑，包括它的开始日期或者是完成日期，是一个时间点，没有持续时间，一般来说给客户看；

横道图（又叫甘特图），不仅仅表现出这个可交付成果的一个时间点，还显示出要做这项活动的开始日期和完成日期，也就是说它可以标明一段时间：什么时候开始，现在进行到什么时间，什么时候会完成。它不是一个时间点，而是一段的持续时间。
甘特图经常用作向管理层来汇报。所以可以显示整个项目的完成百分比和整个项目的进度情况。

逻辑横道图，不仅仅可以知道这个活动什么时候开始，进行到哪一步了，还能知道这个活动大概什么时候能够完成，还知道这个活动做完了下一步要做哪一个活动，所以非常详细。
逻辑横道图经常是项目团队内部使用。

A.里程碑图
B.工作分解结构(WBS)
C.工作绩效数据
D.横道图

【解析：横道图又称甘特图，显示了活动的开始和结束日期。直观易读，常用于向管理层汇报情况。】

2.项目进度计划已准备好批准，项目团队正在制作一份具有重要可交付成果的文件。项目经理正在制作下列哪一分文件？
A.里程碑图
B.甘特图
C.横道图
D.工作分解结构

【解析：主要可交付成果，只有里程碑图能够呈现。】

2019.7.29每日一题：

A new warehouse is ready for operations. Before operations begin, the project manager mustprepare the final project report. What should the project manager refer to?
A. Lessons learned, project charter, and work breakdown structure (WBS)
B. Project baselines,accepted deliverables, and earned value (EV)
C. Checklists, responsibility assignment matrix(RAM), and risk management plan
D. Accepted deliverables, lessons learned, and transition criteria

一个新的仓库准备好运营。运营开始之前，项目经理必须准备最终项目报告。项目经理应该查阅哪些文件？
A.经验教训、项目章程和工作分解结构(WBS)
B.项目基准、已验收的可交付成果和挣值（EV）
C.核对表、责任分配矩阵(RAM）和风险管理计划
D.已验收的可交付成果、经验教训和移交标准

答案：D
解析：四个参考答案中，作为项目转入运营之前的最终报告，最好的内容是参考答案D。
