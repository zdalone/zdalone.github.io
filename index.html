Shell脚本：批量解压

一、需求

将指定目录下的所有tar.gz文件解压并删除压缩文件

二、代码

#! /bin/sh

#需要传入1个参数
if [ $# -ne 1 ];then
echo "USAGE: sh $0 path"
echo "eg: sh $0 /mnt/bigdan1/"
exit 1
fi

#获取1个参数
srcPath=$1

#判断传入的参数是否存在
if [ ! -d "$srcPath" ];then
echo "$srcPath does not exist"
exit 1
fi

REG="*.tar.gz"
#获取srcPath下的所有tar.gz文件
for dir in $(find $srcPath -name $REG)
do
    echo "begin tar -zxf $dir to path: ${dir%/*} `date`"
    tar -zxf $dir -C  ${dir%/*} && rm -rf $dir 
    echo "finish !  `date`"
done

exit 0

三、知识点

① $#：参数个数

② find $path -name $REG : 在$path路径下寻找满足正则$REG的所有文件

③ ${dir%/*} ： 删掉dir最后一个 / 及其右边的字符串

Hexo:安装APlayer音乐播放器

发表于 2019-05-08 | 更新于 2019-05-09 | 分类于 工具 ， Hexo
本文字数： 2.1k | 阅读时长 ≈ 2 分钟

为Hexo安装APlayer音乐播放器

step1

git clone https://github.com/MoePlayer/APlayer.git

step2

cd APlayer
cp -r dist 你的博客主目录/themes/next/source

step3

cd 你的博客主目录/themes/next/source/dist
touch music.js

step4

vim music.js

添加如下内容：

const ap = new APlayer({
container: document.getElementById('aplayer'),
fixed: true,
autoplay: true,
audio: [
  {
    name: "理想三旬",
    artist: '陈鸿宇',
    url: 'http://www.ytmp3.cn/down/46483.mp3',
    cover: 'http://cdnimg103.lizhi.fm/audio_cover/2016/04/12/27782248821326471_320x320.jpg',
  }
]
});

url处填写音乐外链；cover处填写该音乐播放时展示的照片的链接。

step5

vim 你的博客主目录/themes/next/layout/_layout.swig

在<body></body>间添加如下内容：

<link rel="stylesheet" href="/dist/APlayer.min.css">
<div id="aplayer"></div>
<script type="text/javascript" src="/dist/APlayer.min.js"></script>
<script type="text/javascript" src="/dist/music.js">    </script>

step6

cd 你的博客主目录
hexo clean && hexo g && hexo s

使用Myeclipse从GitHub上下载项目

一、准备工作

1、一个github账户

2、Myeclipse2015

3、需要下载到本地的项目地址，如下图：

二、操作步骤

1、右击—>Import—>Git—>Projects from Git—>next

2、选择“Clone URI”->next

3、将项目地址复制到URI中，其他地方会自动填上

4、执行完步骤3后会得到下图：

5、然后输入自己的github账户，勾上store in secure store

6、点击Next

7、点击“Browse”，选择本地代码仓库路径，点击Next

8、这时代码正在下载到本地仓库，需要等待一段时间

9、下载完毕会跳出一个对话框，点击“YES”,之后跳出来几个问题，关了即可。项目代码下载到本地仓库里

10、从本地仓库导入项目到Myeclipse中:import—>Git—>Projects from Git—>next获得如下界面，选择第一个“Existing local repository”

11、选择项目

12、选择第三个选项

13、点击Finish

14、然后就OK了
